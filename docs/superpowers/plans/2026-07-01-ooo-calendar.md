# Out-of-Office Calendar Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Colleagues record out-of-office time on a private calendar and share it with teams and individuals, with month/week/timeline/agenda views, in-app + email notifications, i18n, and timezone awareness.

**Architecture:** Server-first SvelteKit — load functions fetch, form actions mutate (superforms + zod v4), calendar state in query params. Domain tables in Drizzle/Postgres alongside better-auth's generated tables. Custom calendar views built on `@internationalized/date` + shadcn-svelte primitives. Notifications written to a table and emailed via Resend fire-and-forget.

**Tech Stack:** SvelteKit 2 (Svelte 5 runes), Drizzle ORM + Postgres, better-auth 1.4 (email+password), sveltekit-superforms (zod4 adapter), Paraglide JS 2 (en/pl/fr), shadcn-svelte, Tailwind 4, Resend (via plain `fetch`), vitest.

**Spec:** `docs/superpowers/specs/2026-07-01-ooo-calendar-design.md`

## Global Constraints

- Package manager: **pnpm**. Never npm/yarn.
- Svelte 5 **runes mode** is forced project-wide (see `vite.config.ts`).
- Every user-facing string is a Paraglide message with **en, pl, fr** translations (`messages/{locale}.json`, flat snake_case keys). Task 5 creates the full catalog; if a later task needs a missing key, add it to all three files in that task.
- Use existing shadcn-svelte components from `$lib/components/ui/*` wherever possible. Do not hand-roll buttons/dialogs/selects.
- DB: start with `pnpm db:start` (docker, keep running in background), apply schema with `pnpm db:push` (this project uses push, not migration files).
- Verification commands: `pnpm check` (svelte-check), `pnpm test` (vitest run), `pnpm lint`.
- **After writing or editing any `.svelte` file, run the `mcp__svelte__svelte-autofixer` MCP tool on it and fix everything it reports before committing.**
- Conventional commits (`feat:`, `fix:`, `test:`, `docs:`), each ending with the Claude co-author trailer.
- Access control: every load/action re-checks permissions server-side; a calendar the viewer cannot see is a **404**, never 403.
- Timed entries are stored as UTC instants (`timestamptz`) and displayed in the **viewer's** timezone; all-day entries are plain dates and are never timezone-converted.

```ts
import {
	boolean,
	check,
	date,
	index,
	pgEnum,
	pgTable,
	primaryKey,
	text,
	timestamp,
	uniqueIndex,
	uuid
} from 'drizzle-orm/pg-core';
import { user } from './auth.schema';

export * from './auth.schema';

export const oooTypeEnum = pgEnum('ooo_type', [
	'vacation',
	'sick',
	'travel',
	'public_holiday',
	'other'
]);
export const teamRoleEnum = pgEnum('team_role', ['owner', 'member']);
export const notificationTypeEnum = pgEnum('notification_type', ['share_received', 'ooo_added']);

export const team = pgTable('team', {
	id: uuid('id').primaryKey().defaultRandom(),
	name: text('name').notNull(),
	description: text('description'),
	createdAt: timestamp('created_at', { withTimezone: true }).notNull().defaultNow()
});

export const teamMember = pgTable(
	'team_member',
	{
		teamId: uuid('team_id')
			.notNull()
			.references(() => team.id, { onDelete: 'cascade' }),
		userId: text('user_id')
			.notNull()
			.references(() => user.id, { onDelete: 'cascade' }),
		role: teamRoleEnum('role').notNull().default('member'),
		createdAt: timestamp('created_at', { withTimezone: true }).notNull().defaultNow()
	},
	(t) => [primaryKey({ columns: [t.teamId, t.userId] })]
);

export const oooEntry = pgTable(
	'ooo_entry',
	{
		id: uuid('id').primaryKey().defaultRandom(),
		userId: text('user_id')
			.notNull()
			.references(() => user.id, { onDelete: 'cascade' }),
		allDay: boolean('all_day').notNull(),
		// all-day entries: inclusive date range, timezone-independent
		startDate: date('start_date', { mode: 'string' }),
		endDate: date('end_date', { mode: 'string' }),
		// timed entries: UTC instants + owner's zone at creation (for context display)
		startAt: timestamp('start_at', { withTimezone: true }),
		endAt: timestamp('end_at', { withTimezone: true }),
		timezone: text('timezone'),
		type: oooTypeEnum('type').notNull(),
		note: text('note'),
		busyOnly: boolean('busy_only').notNull().default(false),
		createdAt: timestamp('created_at', { withTimezone: true }).notNull().defaultNow(),
		updatedAt: timestamp('updated_at', { withTimezone: true }).notNull().defaultNow()
	},
	(t) => [
		index('ooo_entry_user_idx').on(t.userId),
		check(
			'ooo_entry_shape',
			sql`(${t.allDay} AND ${t.startDate} IS NOT NULL AND ${t.endDate} IS NOT NULL AND ${t.startAt} IS NULL AND ${t.endAt} IS NULL AND ${t.startDate} <= ${t.endDate}) OR (NOT ${t.allDay} AND ${t.startAt} IS NOT NULL AND ${t.endAt} IS NOT NULL AND ${t.timezone} IS NOT NULL AND ${t.startDate} IS NULL AND ${t.endDate} IS NULL AND ${t.startAt} < ${t.endAt})`
		)
	]
);

export const calendarShare = pgTable(
	'calendar_share',
	{
		id: uuid('id').primaryKey().defaultRandom(),
		ownerId: text('owner_id')
			.notNull()
			.references(() => user.id, { onDelete: 'cascade' }),
		sharedWithUserId: text('shared_with_user_id').references(() => user.id, {
			onDelete: 'cascade'
		}),
		sharedWithTeamId: uuid('shared_with_team_id').references(() => team.id, {
			onDelete: 'cascade'
		}),
		createdAt: timestamp('created_at', { withTimezone: true }).notNull().defaultNow()
	},
	(t) => [
		uniqueIndex('calendar_share_user_target').on(t.ownerId, t.sharedWithUserId),
		uniqueIndex('calendar_share_team_target').on(t.ownerId, t.sharedWithTeamId),
		check(
			'calendar_share_one_target',
			sql`(${t.sharedWithUserId} IS NOT NULL) <> (${t.sharedWithTeamId} IS NOT NULL)`
		)
	]
);

export const notification = pgTable(
	'notification',
	{
		id: uuid('id').primaryKey().defaultRandom(),
		userId: text('user_id')
			.notNull()
			.references(() => user.id, { onDelete: 'cascade' }),
		type: notificationTypeEnum('type').notNull(),
		actorId: text('actor_id')
			.notNull()
			.references(() => user.id, { onDelete: 'cascade' }),
		entryId: uuid('entry_id').references(() => oooEntry.id, { onDelete: 'cascade' }),
		readAt: timestamp('read_at', { withTimezone: true }),
		createdAt: timestamp('created_at', { withTimezone: true }).notNull().defaultNow()
	},
	(t) => [index('notification_user_idx').on(t.userId, t.readAt)]
);
```

`src/lib/server/email/send.ts`:

```ts
import { env } from '$env/dynamic/private';

/** Fire-and-forget safe: logs failures, never throws, never blocks the caller's action. */
export async function sendEmail(to: string, subject: string, html: string): Promise<void> {
	if (!env.RESEND_API_KEY || !env.RESEND_EMAIL_ADDRESS) {
		console.warn(`[email] RESEND env vars missing, skipped "${subject}" to ${to}`);
		return;
	}
	try {
		const res = await fetch('https://api.resend.com/emails', {
			method: 'POST',
			headers: {
				Authorization: `Bearer ${env.RESEND_API_KEY}`,
				'Content-Type': 'application/json'
			},
			body: JSON.stringify({ from: env.RESEND_EMAIL_ADDRESS, to, subject, html })
		});
		if (!res.ok) {
			console.error(`[email] send failed (${res.status}): ${await res.text()}`);
		}
	} catch (err) {
		console.error('[email] send failed:', err);
	}
}
```

```ts
import * as m from '$lib/paraglide/messages';
import type { Locale } from '$lib/paraglide/runtime';
import { env } from '$env/dynamic/private';

export function escapeHtml(s: string): string {
	return s
		.replaceAll('&', '&amp;')
		.replaceAll('<', '&lt;')
		.replaceAll('>', '&gt;')
		.replaceAll('"', '&quot;')
		.replaceAll("'", '&#39;');
}

type Email = { subject: string; html: string };

function layout(locale: Locale, ...paragraphs: string[]): string {
	const app = `<p style="margin-top:24px"><a href="${env.ORIGIN || '/'}">${m.email_open_app({}, { locale })}</a></p>`;
	return `<div style="font-family:sans-serif;max-width:480px;margin:0 auto;line-height:1.5">${paragraphs.map((p) => `<p>${p}</p>`).join('')}${app}</div>`;
}

function cta(url: string, label: string): string {
	return `<a href="${url}" style="display:inline-block;background:#18181b;color:#fff;padding:10px 16px;border-radius:6px;text-decoration:none">${label}</a>`;
}

export function verificationEmail(name: string, url: string, locale: Locale): Email {
	return {
		subject: m.email_verify_subject({}, { locale }),
		html: layout(
			locale,
			m.email_greeting({ name: escapeHtml(name) }, { locale }),
			m.email_verify_body({}, { locale }),
			cta(url, m.email_verify_cta({}, { locale }))
		)
	};
}

export function passwordResetEmail(name: string, url: string, locale: Locale): Email {
	return {
		subject: m.email_reset_subject({}, { locale }),
		html: layout(
			locale,
			m.email_greeting({ name: escapeHtml(name) }, { locale }),
			m.email_reset_body({}, { locale }),
			cta(url, m.email_reset_cta({}, { locale }))
		)
	};
}

export function shareReceivedEmail(actorName: string, locale: Locale): Email {
	const name = escapeHtml(actorName);
	return {
		subject: m.email_share_subject({ name: actorName }, { locale }),
		html: layout(locale, m.email_share_body({ name }, { locale }))
	};
}

export function entryAddedEmail(actorName: string, locale: Locale): Email {
	const name = escapeHtml(actorName);
	return {
		subject: m.email_entry_subject({ name: actorName }, { locale }),
		html: layout(locale, m.email_entry_body({ name }, { locale }))
	};
}
```

```ts
import { env } from '$env/dynamic/private';
import { betterAuth } from 'better-auth/minimal';
import { drizzleAdapter } from 'better-auth/adapters/drizzle';
import { sveltekitCookies } from 'better-auth/svelte-kit';
import { getRequestEvent } from '$app/server';
import { db } from '$lib/server/db';
import { sendEmail } from '$lib/server/email/send';
import { passwordResetEmail, verificationEmail } from '$lib/server/email/templates';
import type { Locale } from '$lib/paraglide/runtime';

function userLocale(user: { locale?: string | null }): Locale {
	return (['en', 'pl', 'fr'].includes(user.locale ?? '') ? user.locale : 'en') as Locale;
}

export const auth = betterAuth({
	baseURL: env.ORIGIN,
	secret: env.BETTER_AUTH_SECRET,
	database: drizzleAdapter(db, { provider: 'pg' }),
	user: {
		additionalFields: {
			timezone: { type: 'string', required: false, defaultValue: 'UTC', input: true },
			locale: { type: 'string', required: false, defaultValue: 'en', input: true }
		}
	},
	emailAndPassword: {
		enabled: true,
		requireEmailVerification: true,
		sendResetPassword: async ({ user, url }) => {
			const { subject, html } = passwordResetEmail(user.name, url, userLocale(user));
			await sendEmail(user.email, subject, html);
		}
	},
	emailVerification: {
		sendOnSignUp: true,
		autoSignInAfterVerification: true,
		sendVerificationEmail: async ({ user, url }) => {
			const { subject, html } = verificationEmail(user.name, url, userLocale(user));
			await sendEmail(user.email, subject, html);
		}
	},
	plugins: [
		sveltekitCookies(getRequestEvent) // make sure this is the last plugin in the array
	]
});

export type AuthUser = typeof auth.$Infer.Session.user;
export type AuthSession = typeof auth.$Infer.Session.session;
```

```ts
import { fail, redirect } from '@sveltejs/kit';
import { message, superValidate } from 'sveltekit-superforms';
import { zod4 } from 'sveltekit-superforms/adapters';
import { APIError } from 'better-auth/api';
import { auth } from '$lib/server/auth';
import { loginSchema } from '$lib/schemas/auth';
import * as m from '$lib/paraglide/messages';

export const load = async () => ({ form: await superValidate(zod4(loginSchema)) });

export const actions = {
	default: async ({ request }) => {
		const form = await superValidate(request, zod4(loginSchema));
		if (!form.valid) return fail(400, { form });
		try {
			await auth.api.signInEmail({
				body: { email: form.data.email, password: form.data.password },
				headers: request.headers
			});
		} catch (e) {
			if (e instanceof APIError) {
				const text =
					e.body?.code === 'EMAIL_NOT_VERIFIED'
						? m.auth_email_not_verified()
						: m.auth_invalid_credentials();
				return message(form, text, { status: 'error' });
			}
			throw e;
		}
		redirect(303, '/calendar');
	}
};
```

```ts
export const actions = {
	default: async (event) => {
		const form = await superValidate(event.request, zod4(resetPasswordSchema));

		}

	}
```

```ts
import {
	CalendarDate,
	endOfMonth,
	fromDate,
	parseDate,
	parseDateTime,
	startOfMonth,
	startOfWeek,
	toCalendarDate,
	toZoned
} from '@internationalized/date';
import type { ViewEntry } from './types';

/** Paraglide locales → BCP-47 with region, so week-start resolution is deterministic. */
const BCP47: Record<string, string> = { en: 'en-US', pl: 'pl-PL', fr: 'fr-FR' };
export function bcp47(locale: string): string {
	return BCP47[locale] ?? locale;
}

export function monthGridWeeks(month: CalendarDate, locale: string): CalendarDate[][] {
	const last = endOfMonth(month);
	let day = startOfWeek(startOfMonth(month), bcp47(locale));
	const weeks: CalendarDate[][] = [];
	do {
		const week: CalendarDate[] = [];
		for (let i = 0; i < 7; i++) {
			week.push(day);
			day = day.add({ days: 1 });
		}
		weeks.push(week);
	} while (day.compare(last) <= 0);
	return weeks;
}

export type DateSpan = { start: CalendarDate; end: CalendarDate };

export function entryDateSpan(entry: ViewEntry, viewerTz: string): DateSpan {
	if (entry.allDay) {
		return { start: parseDate(entry.startDate!), end: parseDate(entry.endDate!) };
	}
	return {
		start: toCalendarDate(fromDate(entry.startAt!, viewerTz)),
		end: toCalendarDate(fromDate(entry.endAt!, viewerTz))
	};
}

export function overlapsRange(span: DateSpan, rangeStart: CalendarDate, rangeEnd: CalendarDate): boolean {
	return span.start.compare(rangeEnd) <= 0 && span.end.compare(rangeStart) >= 0;
}

export type LaneSegment = {
	entry: ViewEntry;
	startCol: number;
	span: number;
	continuesLeft: boolean;
	continuesRight: boolean;
	lane: number;
};

/** Lay entries out over a window of consecutive days, stacked greedily into lanes. */
export function laneSegments(entries: ViewEntry[], days: CalendarDate[], viewerTz: string): LaneSegment[] {
	const first = days[0];
	const last = days[days.length - 1];
	const items: Omit<LaneSegment, 'lane'>[] = [];
	for (const entry of entries) {
		const span = entryDateSpan(entry, viewerTz);
		if (!overlapsRange(span, first, last)) continue;
		const start = span.start.compare(first) < 0 ? first : span.start;
		const end = span.end.compare(last) > 0 ? last : span.end;
		const startCol = days.findIndex((d) => d.compare(start) === 0);
		const endCol = days.findIndex((d) => d.compare(end) === 0);
		items.push({
			entry,
			startCol,
			span: endCol - startCol + 1,
			continuesLeft: span.start.compare(first) < 0,
			continuesRight: span.end.compare(last) > 0
		});
	}
	items.sort((a, b) => a.startCol - b.startCol || b.span - a.span);
	const laneEnds: number[] = [];
	return items.map((it) => {
		let lane = laneEnds.findIndex((endCol) => endCol < it.startCol);
		if (lane === -1) {
			lane = laneEnds.length;
			laneEnds.push(-1);
		}
		laneEnds[lane] = it.startCol + it.span - 1;
		return { ...it, lane };
	});
}

export function timedEntryInstants(
	v: { startDate: string; startTime: string; endDate: string; endTime: string },
	timezone: string
): { startAt: Date; endAt: Date } {
	return {
		startAt: toZoned(parseDateTime(`${v.startDate}T${v.startTime}`), timezone).toDate(),
		endAt: toZoned(parseDateTime(`${v.endDate}T${v.endTime}`), timezone).toDate()
	};
}

/** Vertical position of a timed entry within one day column of the week view. */
export function timedBlockRect(
	entry: ViewEntry,
	day: CalendarDate,
	viewerTz: string
): { topPct: number; heightPct: number } | null {
	if (entry.allDay || !entry.startAt || !entry.endAt) return null;
	const dayStart = day.toDate(viewerTz).getTime();
	const dayEnd = day.add({ days: 1 }).toDate(viewerTz).getTime();
	const start = Math.max(entry.startAt.getTime(), dayStart);
	const end = Math.min(entry.endAt.getTime(), dayEnd);
	if (end <= start) return null;
	const total = dayEnd - dayStart;
	return { topPct: ((start - dayStart) / total) * 100, heightPct: ((end - start) / total) * 100 };
}

/** Busy-only entries hide type and note from everyone but the owner. */
export function sanitizeForViewer(entry: ViewEntry, viewerId: string): ViewEntry {
	if (!entry.busyOnly || entry.userId === viewerId) return entry;
	return { ...entry, type: null, note: null };
}
```

```ts
export function shareRecipientIds(
	ownerId: string,
	directUserIds: string[],
	teamMemberIds: string[]
): string[] {
	return [...new Set([...directUserIds, ...teamMemberIds])].filter((id) => id !== ownerId);
}

type Member = { userId: string; role: 'owner' | 'member' };

export function canRemoveMember(members: Member[], actorId: string, targetId: string): boolean {
	const actor = members.find((m) => m.userId === actorId);
	const target = members.find((m) => m.userId === targetId);
	if (!actor || !target) return false;
	if (actorId !== targetId && actor.role !== 'owner') return false;
	const ownerCount = members.filter((m) => m.role === 'owner').length;
	if (target.role === 'owner' && ownerCount === 1) return false;
	return true;
}

export function startsWithinDays(
	entry: { allDay: boolean; startDate: string | null; startAt: Date | null },
	days: number,
	now: Date
): boolean {
	const limit = new Date(now.getTime() + days * 86_400_000);
	if (entry.allDay) {
		return entry.startDate !== null && entry.startDate <= limit.toISOString().slice(0, 10);
	}
	return entry.startAt !== null && entry.startAt.getTime() <= limit.getTime();
}
```

```ts
import { entryDateSpan, overlapsRange, sanitizeForViewer, timedEntryInstants } from '$lib/calendar/utils';
import type { EntryInput } from '$lib/schemas/ooo';

function entryColumns(input: EntryInput, ownerTz: string) {
	const shared = { type: input.type, note: input.note || null, busyOnly: input.busyOnly };
	if (input.allDay) {
		return {
			...shared,
			allDay: true,
			startDate: input.startDate,
			endDate: input.endDate,
			startAt: null,
			endAt: null,
			timezone: null

			startTime: input.startTime!,
			endDate: input.endDate,
			endTime: input.endTime!
		},

	return { ...shared, allDay: false, startDate: null, endDate: null, startAt, endAt, timezone: ownerTz };

export async function createEntry(ownerId: string, ownerTz: string, input: EntryInput) {

		.insert(oooEntry)
		.values({ userId: ownerId, ...entryColumns(input, ownerTz) })

			id: oooEntry.id,
			allDay: oooEntry.allDay,
			startDate: oooEntry.startDate,
			startAt: oooEntry.startAt
		});

export async function updateEntry(entryId: string, ownerId: string, ownerTz: string, input: EntryInput): Promise<boolean> {
	const rows = await db
		.update(oooEntry)
		.set({ ...entryColumns(input, ownerTz), updatedAt: new Date() })
		.where(and(eq(oooEntry.id, entryId), eq(oooEntry.userId, ownerId)))
		.returning({ id: oooEntry.id });
	return rows.length > 0;
}

export async function deleteEntry(entryId: string, ownerId: string): Promise<boolean> {
	const rows = await db
		.delete(oooEntry)
		.where(and(eq(oooEntry.id, entryId), eq(oooEntry.userId, ownerId)))
		.returning({ id: oooEntry.id });
	return rows.length > 0;
}

/** Entries of `ownerIds` overlapping [rangeStart, rangeEnd] as seen by `viewerId`, sorted by start. */
export async function entriesForOwnersInRange(
	ownerIds: string[],
	rangeStart: CalendarDate,
	rangeEnd: CalendarDate,
	viewerId: string,
	viewerTz: string
): Promise<ViewEntry[]> {
	if (!ownerIds.length) return [];
	// Coarse SQL filter (±2-day margin absorbs timezone skew), precise filter in JS below.
	const startMargin = rangeStart.subtract({ days: 2 }).toDate(viewerTz);
	const endMargin = rangeEnd.add({ days: 2 }).toDate(viewerTz);
	const rows = await db
		.select()
		.from(oooEntry)
		.where(
			and(
				inArray(oooEntry.userId, ownerIds),
				or(
					and(
						eq(oooEntry.allDay, true),
						lte(oooEntry.startDate, rangeEnd.toString()),
						gte(oooEntry.endDate, rangeStart.toString())
					),
					and(
						eq(oooEntry.allDay, false),
						lte(oooEntry.startAt, endMargin),
						gte(oooEntry.endAt, startMargin)
					)
				)
			)
		);
	return rows
		.map((r) => r as ViewEntry)
		.filter((e) => overlapsRange(entryDateSpan(e, viewerTz), rangeStart, rangeEnd))
		.map((e) => sanitizeForViewer(e, viewerId))
		.sort((a, b) => entryDateSpan(a, viewerTz).start.compare(entryDateSpan(b, viewerTz).start));
}
```

```ts
const PRUNE_AFTER_MS = 90 * 86_400_000;
type Actor = { id: string; name: string };
type NotificationType = 'share_received' | 'ooo_added';

function toLocale(value: string | null | undefined): Locale {
	return (value === 'pl' || value === 'fr' ? value : 'en') as Locale;
}

async function createAndEmail(type: NotificationType, actor: Actor, recipientIds: string[], entryId?: string) {
	if (!recipientIds.length) return;
	// Opportunistic prune per spec: rows older than 90 days go on write.
	await db.delete(notification).where(lt(notification.createdAt, new Date(Date.now() - PRUNE_AFTER_MS)));
	await db
		.insert(notification)
		.values(recipientIds.map((userId) => ({ userId, type, actorId: actor.id, entryId: entryId ?? null })));
	const recipients = await db
		.select({ email: user.email, locale: user.locale })
		.from(user)
		.where(inArray(user.id, recipientIds));
	for (const r of recipients) {
		const locale = toLocale(r.locale);
		const { subject, html } =
			type === 'share_received' ? shareReceivedEmail(actor.name, locale) : entryAddedEmail(actor.name, locale);
		void sendEmail(r.email, subject, html); // fire-and-forget: never blocks or fails the action
	}
}

export async function notifyShareCreated(actor: Actor, recipientIds: string[]): Promise<void> {
	await createAndEmail('share_received', actor, recipientIds);
}

export async function notifyEntryCreated(
	actor: Actor,
	entry: { id: string; allDay: boolean; startDate: string | null; startAt: Date | null }
): Promise<void> {
	if (!startsWithinDays(entry, 60, new Date())) return;
	const recipientIds = await viewerIdsOfCalendar(actor.id);
	await createAndEmail('ooo_added', actor, recipientIds, entry.id);
}

export async function unreadCount(userId: string): Promise<number> {
	const [row] = await db
		.select({ n: count() })
		.from(notification)
		.where(and(eq(notification.userId, userId), isNull(notification.readAt)));
	return row?.n ?? 0;
}

export async function recentNotifications(userId: string, limit = 15) {
	return db
		.select({
			id: notification.id,
			type: notification.type,
			actorId: notification.actorId,
			actorName: user.name,
			readAt: notification.readAt,
			createdAt: notification.createdAt
		})
		.from(notification)
		.innerJoin(user, eq(user.id, notification.actorId))
		.where(eq(notification.userId, userId))
		.orderBy(desc(notification.createdAt))
		.limit(limit);
}

export async function markAllRead(userId: string): Promise<void> {
	await db
		.update(notification)
		.set({ readAt: new Date() })
		.where(and(eq(notification.userId, userId), isNull(notification.readAt)));
}
```

```ts
import type { ViewEntry } from '$lib/calendar/types';
import { bcp47, entryDateSpan } from '$lib/calendar/utils';
import * as m from '$lib/paraglide/messages';

/** "Jul 14 – 18" for all-day, "Jul 14, 09:00 – 13:00" for timed — in the viewer's zone. */
export function formatEntryRange(entry: ViewEntry, viewerTz: string, locale: string): string {
	const tag = bcp47(locale);
	if (entry.allDay) {
		const span = entryDateSpan(entry, viewerTz);
		const fmt = new Intl.DateTimeFormat(tag, { day: 'numeric', month: 'short', timeZone: 'UTC' });
		return fmt.formatRange(span.start.toDate('UTC'), span.end.toDate('UTC'));
	}
	const fmt = new Intl.DateTimeFormat(tag, {
		day: 'numeric',
		month: 'short',
		hour: '2-digit',
		minute: '2-digit',
		timeZone: viewerTz
	});
	return fmt.formatRange(entry.startAt!, entry.endAt!);
}

/** "15:00 – 19:00 in Europe/Warsaw" when the owner's zone differs from the viewer's, else null. */
export function ownerZoneContext(entry: ViewEntry, viewerTz: string, locale: string): string | null {
	if (entry.allDay || !entry.timezone || entry.timezone === viewerTz) return null;
	const fmt = new Intl.DateTimeFormat(bcp47(locale), {
		hour: '2-digit',
		minute: '2-digit',
		timeZone: entry.timezone
	});
	return m.entry_owner_time({
		time: fmt.formatRange(entry.startAt!, entry.endAt!),
		zone: entry.timezone
	});
}

export function formatMonthTitle(date: Date, locale: string): string {
	return new Intl.DateTimeFormat(bcp47(locale), { month: 'long', year: 'numeric', timeZone: 'UTC' }).format(date);
}

export function formatWeekday(date: Date, locale: string): string {
	return new Intl.DateTimeFormat(bcp47(locale), { weekday: 'short', timeZone: 'UTC' }).format(date);
```

```ts
import { fail } from '@sveltejs/kit';
import { redirect } from 'sveltekit-flash-message/server';
import { message, superValidate } from 'sveltekit-superforms';
import { zod4 } from 'sveltekit-superforms/adapters';
import { endOfMonth, parseDate, startOfMonth, today } from '@internationalized/date';
import { getLocale } from '$lib/paraglide/runtime';
import * as m from '$lib/paraglide/messages';
import { entrySchema } from '$lib/schemas/ooo';
import { createEntry, deleteEntry, entriesForOwnersInRange, updateEntry } from '$lib/server/ooo';
import { notifyEntryCreated } from '$lib/server/notifications';

const VIEWS = ['month', 'week', 'agenda'] as const;
type View = (typeof VIEWS)[number];

function parseView(raw: string | null): View {
	return (VIEWS as readonly string[]).includes(raw ?? '') ? (raw as View) : 'month';
}

function parseDateParam(raw: string | null, tz: string) {
	if (raw) {
		try {
			return parseDate(raw);
		} catch {
			// fall through to today
		}
	}
	return today(tz);
}

export const load = async ({ locals, url, parent }) => {
	const { user } = await parent();
	const view = parseView(url.searchParams.get('view'));
	const date = parseDateParam(url.searchParams.get('date'), user.timezone);
	const range =
		view === 'week'
			? { start: date.subtract({ days: 7 }), end: date.add({ days: 13 }) }
			: view === 'agenda'
				? { start: today(user.timezone), end: today(user.timezone).add({ days: 90 }) }
				: { start: startOfMonth(date).subtract({ days: 7 }), end: endOfMonth(date).add({ days: 7 }) };
	const entries = await entriesForOwnersInRange([user.id], range.start, range.end, user.id, user.timezone);
	return {
		view,
		date: date.toString(),
		locale: getLocale(),
		entries,
		form: await superValidate(zod4(entrySchema))
	};
};

export const actions = {
	save: async (event) => {
		const user = event.locals.user!;
		const form = await superValidate(event.request, zod4(entrySchema));
		if (!form.valid) return fail(400, { form });
		if (form.data.id) {
			const ok = await updateEntry(form.data.id, user.id, user.timezone, form.data);
			if (!ok) return message(form, m.error_generic(), { status: 'error' });
		} else {
			const row = await createEntry(user.id, user.timezone, form.data);
			await notifyEntryCreated({ id: user.id, name: user.name }, row);
		}
		redirect(calendarUrl(event.url), { type: 'success', message: m.entry_saved() }, event.cookies);
	},
	delete: async (event) => {
		const user = event.locals.user!;
		const id = (await event.request.formData()).get('id');
		if (typeof id === 'string' && id) await deleteEntry(id, user.id);
		redirect(calendarUrl(event.url), { type: 'success', message: m.entry_deleted() }, event.cookies);
	}
};

// The action URL carries the `?/save` key; rebuild only view/date so the
// redirect returns to the same calendar position with a clean URL.
function calendarUrl(url: URL): string {
	const params = new URLSearchParams();
	for (const key of ['view', 'date']) {
		const value = url.searchParams.get(key);
		if (value) params.set(key, value);
	}
	return `/calendar?${params.toString()}`;
}
```

```ts
import type { CalendarDate as CD } from '@internationalized/date';

let dragAnchor = $state<CD | null>(null);
let dragCurrent = $state<CD | null>(null);

const dragRange = $derived.by(() => {
	if (!dragAnchor || !dragCurrent) return null;
	return dragAnchor.compare(dragCurrent) <= 0
		? { start: dragAnchor, end: dragCurrent }
		: { start: dragCurrent, end: dragAnchor };
});

function inDragRange(day: CD): boolean {
	return dragRange !== null && day.compare(dragRange.start) >= 0 && day.compare(dragRange.end) <= 0;
}

function pointerDown(day: CD, e: PointerEvent) {
	if (!editable || e.button !== 0) return;
	dragAnchor = day;
	dragCurrent = day;
}

function pointerEnter(day: CD) {
	if (dragAnchor) dragCurrent = day;
}

function pointerUp() {
	if (dragAnchor && dragRange) {
		onSelectRange?.(dragRange.start.toString(), dragRange.end.toString());
	}
	dragAnchor = null;
	dragCurrent = null;
}
```

```ts
import { error } from '@sveltejs/kit';
import { eq } from 'drizzle-orm';
import { endOfMonth, parseDate, startOfMonth, today } from '@internationalized/date';
import { getLocale } from '$lib/paraglide/runtime';
import { db } from '$lib/server/db';
import { user as userTable } from '$lib/server/db/schema';
import { canViewCalendar } from '$lib/server/shares';
import { entriesForOwnersInRange } from '$lib/server/ooo';

const VIEWS = ['month', 'week', 'agenda'] as const;

export const load = async ({ params, url, parent }) => {
	const { user } = await parent();
	// 404, not 403: never leak whether the calendar exists.
	if (!(await canViewCalendar(user.id, params.userId))) error(404);
	const [owner] = await db
		.select({ id: userTable.id, name: userTable.name })
		.from(userTable)
		.where(eq(userTable.id, params.userId));
	if (!owner) error(404);

	const rawView = url.searchParams.get('view') ?? 'month';
	const view = (VIEWS as readonly string[]).includes(rawView) ? (rawView as (typeof VIEWS)[number]) : 'month';
	let date = today(user.timezone);
	const rawDate = url.searchParams.get('date');
	if (rawDate) {
		try {
			date = parseDate(rawDate);
		} catch {
			// keep today
		}
	}
	const range =
		view === 'week'
			? { start: date.subtract({ days: 7 }), end: date.add({ days: 13 }) }
			: view === 'agenda'
				? { start: today(user.timezone), end: today(user.timezone).add({ days: 90 }) }
				: { start: startOfMonth(date).subtract({ days: 7 }), end: endOfMonth(date).add({ days: 7 }) };
	const entries = await entriesForOwnersInRange([owner.id], range.start, range.end, user.id, user.timezone);
	return { owner, view, date: date.toString(), locale: getLocale(), entries };
};
```

```ts
import { error, fail } from '@sveltejs/kit';
import { redirect, setFlash } from 'sveltekit-flash-message/server';
import { superValidate } from 'sveltekit-superforms';
import { zod4 } from 'sveltekit-superforms/adapters';
import { parseDate, today } from '@internationalized/date';
import { getLocale } from '$lib/paraglide/runtime';
import * as m from '$lib/paraglide/messages';
import { addMemberSchema, teamSchema } from '$lib/schemas/team';
import { entriesForOwnersInRange } from '$lib/server/ooo';
import { canViewCalendar } from '$lib/server/shares';
import {
	addMemberByEmail,
	deleteTeam,
	getTeamWithMembers,
	memberRole,
	removeMember,
	updateTeam
} from '$lib/server/teams';

const WINDOW_DAYS = 14;

export const load = async ({ params, url, parent }) => {
	const { user } = await parent();
	const role = await memberRole(params.teamId, user.id);
	if (!role) error(404);
	const team = await getTeamWithMembers(params.teamId);
	if (!team) error(404);

	let start = today(user.timezone).subtract({ days: 2 });
	const rawStart = url.searchParams.get('start');
	if (rawStart) {
		try {
			start = parseDate(rawStart);
		} catch {
			// keep default
		}
	}
	const days = Array.from({ length: WINDOW_DAYS }, (_, i) => start.add({ days: i }));
	const end = days[days.length - 1];

	// The timeline shows only what each member's sharing rules let *this viewer* see.
	const visibleIds: string[] = [];
	for (const member of team.members) {
		if (await canViewCalendar(user.id, member.userId)) visibleIds.push(member.userId);
	}
	const entries = await entriesForOwnersInRange(visibleIds, start, end, user.id, user.timezone);
	const byOwner = new Map<string, typeof entries>();
	for (const e of entries) byOwner.set(e.userId, [...(byOwner.get(e.userId) ?? []), e]);
	const rows = team.members.map((mem) => ({
		user: { id: mem.userId, name: mem.name },
		entries: byOwner.get(mem.userId) ?? []
	}));

	return {
		team,
		role,
		rows,
		start: start.toString(),
		windowDays: WINDOW_DAYS,
		locale: getLocale(),
		addMemberForm: await superValidate(zod4(addMemberSchema)),
		editTeamForm: await superValidate(
			{ name: team.name, description: team.description ?? undefined },
			zod4(teamSchema)
		)
	};
};

export const actions = {
	addMember: async (event) => {
		const user = event.locals.user!;
		if ((await memberRole(event.params.teamId, user.id)) !== 'owner') error(404);
		const form = await superValidate(event.request, zod4(addMemberSchema));
		if (!form.valid) return fail(400, { form });
		const result = await addMemberByEmail(event.params.teamId, form.data.email);
		const msg =
			result === 'added'
				? ({ type: 'success', message: m.team_member_added() } as const)
				: result === 'already'
					? ({ type: 'error', message: m.team_member_already() } as const)
					: ({ type: 'error', message: m.team_member_not_found() } as const);
		redirect(`/teams/${event.params.teamId}`, msg, event.cookies);
	},
	removeMember: async (event) => {
		const user = event.locals.user!;
		if (!(await memberRole(event.params.teamId, user.id))) error(404);
		const targetId = (await event.request.formData()).get('userId');
		if (typeof targetId !== 'string') return fail(400);
		const result = await removeMember(event.params.teamId, user.id, targetId);
		if (result === 'forbidden') {
			setFlash({ type: 'error', message: m.team_last_owner() }, event.cookies);
			return fail(403);
		}
		// Leaving the team yourself: you can no longer see it.
		const dest = targetId === user.id ? '/teams' : `/teams/${event.params.teamId}`;
		redirect(dest, { type: 'success', message: m.team_member_removed() }, event.cookies);
	},
	updateTeam: async (event) => {
		const user = event.locals.user!;
		if ((await memberRole(event.params.teamId, user.id)) !== 'owner') error(404);
		const form = await superValidate(event.request, zod4(teamSchema));
		if (!form.valid) return fail(400, { form });
		await updateTeam(event.params.teamId, form.data.name, form.data.description ?? null);
		redirect(`/teams/${event.params.teamId}`, { type: 'success', message: m.team_updated() }, event.cookies);
	},
	deleteTeam: async (event) => {
		const user = event.locals.user!;
		if ((await memberRole(event.params.teamId, user.id)) !== 'owner') error(404);
		await deleteTeam(event.params.teamId);
		redirect('/teams', { type: 'success', message: m.team_deleted() }, event.cookies);
	}
};
```

```ts
import { fail } from '@sveltejs/kit';
import { redirect } from 'sveltekit-flash-message/server';
import { message, superValidate } from 'sveltekit-superforms';
import { zod4 } from 'sveltekit-superforms/adapters';
import { eq } from 'drizzle-orm';
import { APIError } from 'better-auth/api';
import * as m from '$lib/paraglide/messages';
import { auth } from '$lib/server/auth';
import { db } from '$lib/server/db';
import { user as userTable } from '$lib/server/db/schema';
import { changePasswordSchema, profileSchema, shareTeamSchema, shareUserSchema } from '$lib/schemas/settings';
import { notifyShareCreated } from '$lib/server/notifications';
import { createTeamShare, createUserShare, deleteShare, listOutgoingShares } from '$lib/server/shares';
import { getTeamWithMembers, listMyTeams, memberRole } from '$lib/server/teams';
import { shareRecipientIds } from '$lib/server/rules';

export const load = async ({ parent }) => {
	const { user } = await parent();
	return {
		shares: await listOutgoingShares(user.id),
		myTeams: await listMyTeams(user.id),
		profileForm: await superValidate(
			{ name: user.name, timezone: user.timezone, locale: user.locale as 'en' | 'pl' | 'fr' },
			zod4(profileSchema)
		),
		passwordForm: await superValidate(zod4(changePasswordSchema)),
		shareUserForm: await superValidate(zod4(shareUserSchema)),
		shareTeamForm: await superValidate(zod4(shareTeamSchema))
	};
};

export const actions = {
	profile: async (event) => {
		const form = await superValidate(event.request, zod4(profileSchema));
		if (!form.valid) return fail(400, { form });
		await auth.api.updateUser({
			body: { name: form.data.name, timezone: form.data.timezone, locale: form.data.locale },
			headers: event.request.headers
		});
		event.cookies.set('PARAGLIDE_LOCALE', form.data.locale, { path: '/' });
		redirect('/settings', { type: 'success', message: m.settings_profile_saved() }, event.cookies);
	},
	password: async (event) => {
		const form = await superValidate(event.request, zod4(changePasswordSchema));
		if (!form.valid) return fail(400, { form });
		try {
			await auth.api.changePassword({
				body: {
					currentPassword: form.data.currentPassword,
					newPassword: form.data.newPassword,
					revokeOtherSessions: true
				},
				headers: event.request.headers
			});
		} catch (e) {
			if (e instanceof APIError) return message(form, m.settings_password_wrong(), { status: 'error' });
			throw e;
		}
		redirect('/settings', { type: 'success', message: m.settings_password_changed() }, event.cookies);
	},
	shareUser: async (event) => {
		const user = event.locals.user!;
		const form = await superValidate(event.request, zod4(shareUserSchema));
		if (!form.valid) return fail(400, { form });
		const [target] = await db
			.select({ id: userTable.id })
			.from(userTable)
			.where(eq(userTable.email, form.data.email));
		if (!target || target.id === user.id) return message(form, m.settings_share_user_not_found(), { status: 'error' });
		const result = await createUserShare(user.id, target.id);
		if (result === 'exists') return message(form, m.settings_share_already(), { status: 'error' });
		await notifyShareCreated({ id: user.id, name: user.name }, [target.id]);
		redirect('/settings', { type: 'success', message: m.settings_share_added() }, event.cookies);
	},
	shareTeam: async (event) => {
		const user = event.locals.user!;
		const form = await superValidate(event.request, zod4(shareTeamSchema));
		if (!form.valid) return fail(400, { form });
		// Only teams the user belongs to are valid share targets.
		if (!(await memberRole(form.data.teamId, user.id))) return fail(403, { form });
		const result = await createTeamShare(user.id, form.data.teamId);
		if (result === 'exists') return message(form, m.settings_share_already(), { status: 'error' });
		const team = await getTeamWithMembers(form.data.teamId);
		const recipients = shareRecipientIds(user.id, [], team?.members.map((mem) => mem.userId) ?? []);
		await notifyShareCreated({ id: user.id, name: user.name }, recipients);
		redirect('/settings', { type: 'success', message: m.settings_share_added() }, event.cookies);
	},
	unshare: async (event) => {
		const user = event.locals.user!;
		const id = (await event.request.formData()).get('id');
		if (typeof id === 'string' && id) await deleteShare(id, user.id);
		redirect('/settings', { type: 'success', message: m.settings_share_removed() }, event.cookies);
	}
};
```
